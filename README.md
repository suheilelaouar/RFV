# RFV
Projeto de exemplo para aplicação de segmentação de cientes com RFV

## Descrição do projeto
No link abaixo, é possível adicionar um arquivo com as colunas ID_cliente,CodigoCompra,DiaCompra,ValorTotal.
A análise dos dados é realizada e a segmentação criada com base nos comportamentos de compra do cliante (Recência, FrequÊncia e Valor)
